🧩 4. Trelay – Full Stack Developer & Systems Engineer
The Challenge
Most URL shorteners are either overly simplistic or tied to centralized services, limiting control, extensibility, and privacy.
Developers who want self-hosted solutions often have to compromise on usability, automation, or ecosystem integration.
The Solution
I built a developer-first, self-hosted URL management system that combines a modern dashboard, CLI, and API into a single cohesive platform.
Key decisions:
Designed a unified system with dashboard + CLI + API parity
Built automation-friendly workflows for scripting and integrations
Focused on privacy and control with self-hosted deployment
Implemented structured features like expiring links, one-time access, analytics, and custom routing
The system is packaged as a single-container deployment, making it easy to run while still being production-ready.
The Impact
Trelay enables developers to manage links as part of their infrastructure rather than relying on third-party tools.
It demonstrates how developer tooling can be both powerful and accessible, without sacrificing control or simplicity.
Links
Live: https://trelay.dev
GitHub: https://github.com/trelay-dev/trelay
